# Simulator examen licență

Aplicație statică HTML/CSS/JavaScript.

## Moduri incluse

1. Simulare examen: 45 întrebări, 2 puncte fiecare + 10 puncte din oficiu.
2. Practică pe categorii: fără punctaj, cu feedback instant.
3. Învățare pe categorii: meniu de navigare prin fiecare categorie, întrebări, variante și răspuns corect afișat.

## Publicare online

Poate fi publicată gratuit pe GitHub Pages, Netlify sau Vercel. Pentru GitHub Pages, fișierele `index.html`, `style.css`, `script.js` și `questions.js` trebuie să fie direct în rădăcina repository-ului.
